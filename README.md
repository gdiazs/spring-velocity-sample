# spring-velocity-sample
A project for test Apache Velocity (just for have fun)

Requeriments:
-------------

Java 1.8
Maven 3.x

JAVA_HOME and M2_HOME configured

Compile with: 

    mvn package or mvn install

Run Server

     jetty:run
